영화, TV 프로그램 등 컨텐츠 정보를 개봉순, 평점순, 인기순 등으로 조회하고 검색할 수 있습니다.

## 기능

### TMDB API 데이터 활용

`TMDB API`를 호출해 화면에 데이터를 시각적으로 구현했습니다.

### 컨텐츠 목록 슬라이드

`React Slick` 라이브러리를 사용해 컨텐츠 목록 슬라이더를 구현했습니다.

### 검색

컨텐츠를 검색할 수 있습니다. 검색 결과는 영화, TV 프로그램에 따라 분류되어 보여집니다.
